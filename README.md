## ♻️ Como Contribuir

```bash
  # Clone the project
  $ git clone https://github.com/Japolina/CRUD_Tarefas.git
```

```bash
  # Enter directory
  $ cd PRUPC
```

```bash
  # Install the dependencies, if you use npm
  $ npm install
```

```bash

  $ yarn
```
